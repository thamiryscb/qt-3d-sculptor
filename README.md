# Voxel Sculptor – C++ / Qt

Este projeto consiste em um **escultor 3D baseado em voxels**, desenvolvido em **C++ utilizando o framework Qt**, como parte de um trabalho acadêmico da disciplina de Programação Avançada.

A aplicação permite a criação e manipulação de objetos tridimensionais por meio de voxels, aplicando conceitos de **programação orientada a objetos**, estruturas de dados e representação gráfica.

---

## Objetivos do Projeto

- Desenvolver um escultor 3D baseado em voxels
- Aplicar conceitos de Programação Orientada a Objetos (POO) em C++
- Manipular estruturas de dados tridimensionais
- Implementar a criação e remoção de voxels
- Gerar modelos 3D de forma programática

---

## Conceitos Aplicados

- Programação Orientada a Objetos (POO)
- Encapsulamento e modularização
- Alocação dinâmica de memória
- Matrizes tridimensionais (voxels)
- Manipulação de arquivos
- Organização de código em múltiplos arquivos

---

## Estrutura do Projeto

- `main.cpp` – Ponto de entrada da aplicação
- `sculptor.h / sculptor.cpp` – Lógica principal do escultor 3D
- `voxel.h` – Definição da estrutura de um voxel
- Arquivos `.pro` – Configuração do projeto no Qt
- Pasta `build/` – Diretório de compilação (gerado pelo Qt)

---

##  Tecnologias Utilizadas

- Linguagem C++
- Framework Qt
- Qt Creator
